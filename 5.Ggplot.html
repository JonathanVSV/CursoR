5.Ggplot

Autor/a

Jonathan V. Solórzano

Fecha de publicación

3 de octubre de 2024

Paquete para hacer gráficos

Este paquete está enfocado en programar gráficas.

Ventajas:

  • 100 % editable.
  • Permite crear prácticamente cualquier tipo de gráfica.
  • Varios paquetes usar como base ggplot2 para generar sus propios gráficos.

Los gráficos de ggplot normalmente siguen la siguiente estructura:

  1. Datos.
  2. Mapeo de variables estéticas aes.
  3. Adición de capas.

Leer mismos datos

df <- read_excel("Data/INFyS_2015_2020_Michoacan_de_Ocampo_lHwLKIM.xlsx",
                  sheet = "Arbolado")
Warning: Expecting numeric in BW19313 / R19313C75: got 'NULL'

Como funciona ggplot2 es como si fueran capas similares a como se manejan imágenes en Photoshop o GIMP.

La primera parte genera el espacio o la base del gráfico.

df |>
  filter(DiametroBasalSub_validacion_C3 <= 100000) |>
  ggplot(aes(x = DiametroBasalSub_validacion_C3,
             y = biomasa_kg_C3))

Tipos de gráficos

Después se define el estilo de gráfico que se desea.

Scatterplot

df |>
  filter(DiametroBasalSub_validacion_C3 <= 100000) |>
  ggplot(aes(x = DiametroBasalSub_validacion_C3,
             y = biomasa_kg_C3,)) +
  geom_point()

Las opciones básicas de tipos de gráficos son:

  • geom_point.
  • geom_area.
  • geom_bar (frecuencias).
  • geom_col (alguna otra variable).
  • geom_boxplot.
  • geom_errorbar
  • geom_density.
  • geom_histogram.
  • geom_line.
  • geom_smooth.
  • geom_line.

Aunque hay muchas más, veamos algunos ejemplos.

Aes guías

Guía para definir parámetros de aes:

  • x: valores en eje x.
  • y: valores en eje y.
  • col: color de bordes o líneas.
  • fill: relleno de barras, áreas, etc.
  • group: Grupos, p.ej., para series de puntos y líneas, unir las de un mismo grupo

Ahorita ya usamos x y y, veamos col y fill.

df |>
  filter(DiametroBasalSub_validacion_C3 <= 100000) |>

Scales

  • Scale_x
    • _continuous: cuando la variable es numérica.
    • _discrete: cuando la variable es discreta (factor).
  • Scale_y
    • _continuous: cuando la variable es numérica.
    • _discrete: cuando la variable es discreta (factor).
df |>
  filter(DiametroBasalSub_validacion_C3 <= 100000) |>

             y = biomasa_kg_C3,
             col = CVE_S7_C3,
             group = CVE_S7_C3)) +
  geom_line() +
  scale_x_continuous(breaks = seq(0,250,25),
                     limits = c(0,250),
                     expand = c(0,0))+
  scale_y_continuous(breaks = seq(0,10000,1250),
                     limits = c(0,10000),
                     expand = c(0,0))

Facets

Facet_wrap

Hacer gráficos de varios paneles.

df |>
  filter(DiametroBasalSub_validacion_C3 <= 100000) |>
  group_by(Anio_C3, CVE_S7_C3) |>
  summarise(mean = mean(biomasa_kg_C3, na.rm = TRUE),
            sd = sd(biomasa_kg_C3, na.rm = TRUE),
            .groups = "drop") |>
  ggplot(aes(x = CVE_S7_C3,
             y = mean,
             fill = as.factor(Anio_C3))) +
  geom_col() +
  facet_wrap(~ as.factor(Anio_C3),
             ncol = 3,
             scales = "free_y")

Facet_grid

Hacer gráficos de varios paneles, con una cuadrícula de dos factores

df |>
  filter(DiametroBasalSub_validacion_C3 <= 100000) |>
  group_by(Anio_C3, CVEECON1_C3, CVE_S7_C3) |>
  summarise(mean = mean(biomasa_kg_C3, na.rm = TRUE),
            sd = sd(biomasa_kg_C3, na.rm = TRUE),
            .groups = "drop") |>
  ggplot(aes(x = CVE_S7_C3,
             y = mean,
             fill = as.factor(Anio_C3))) +
  geom_col() +
  facet_grid(as.factor(CVEECON1_C3) ~ as.factor(Anio_C3),
             scales = "free_y")

Comparado con facet_wrap:

df |>
  filter(DiametroBasalSub_validacion_C3 <= 100000) |>
  group_by(Anio_C3, CVEECON1_C3, CVE_S7_C3) |>
  summarise(mean = mean(biomasa_kg_C3, na.rm = TRUE),
            sd = sd(biomasa_kg_C3, na.rm = TRUE),
            .groups = "drop") |>
  ggplot(aes(x = CVE_S7_C3,
             y = mean,
             fill = as.factor(Anio_C3))) +
  geom_col() +
  facet_wrap(as.factor(CVEECON1_C3) ~ as.factor(Anio_C3),
             scales = "free_y")

Personalizar tema

Opciones de tema tenemos opciones sobre:

  • Ejes.
    • Títulos de ejes.
    • Texto de ejes.
    • Marcas de ejes.
  • Leyenda.
    • Texto.
    • Fondo.
    • Borde.
  • Panel.
    • Fondo.
    • Divisiones menores y mayores.
  • Área de gráfica.
    • Título.
    • Fondo.
  • Tiras (Strip)
    • Texto.
    • Fondo.

Ejemplo

df |>
  filter(DiametroBasalSub_validacion_C3 <= 100000) |>
  group_by(CVE_S7_C3) |>
  summarise(mean = mean(biomasa_kg_C3, na.rm = TRUE),
            sd = sd(biomasa_kg_C3, na.rm = TRUE), 
            .groups = "drop") |>
  mutate(largocad = str_length(CVE_S7_C3)) |>
  filter(largocad <= 6) |>
  ggplot(aes(x = CVE_S7_C3,
             y = mean)) +
  geom_col() +
  theme(plot.title = element_text(size=18,hjust = 0.5),
        text = element_text(size=24,colour="black"),
        axis.text.x = element_text(size=18,
                                   colour="black",
                                   angle = 90, 
                                   hjust = 1,
                                   vjust = 0.5),
        axis.text.y = element_text(size=18,
                                   colour="black",
                                   angle = 0, 
                                   vjust = 0.5,
                                   hjust = 1),
        axis.title = element_text(size=18,
                                  colour="black",
                                  face = "bold"), 
        axis.line = element_line(colour = "black"),
        legend.title = element_text(size=18),
        legend.text = element_text(size=18),
        axis.line.x = element_line(colour="black"),
        axis.line.y = element_line(colour="black"),
        panel.grid.major = element_blank(),
        panel.grid.minor = element_blank(),
        panel.border = element_blank(),
        panel.background = element_blank(),
        strip.background = element_rect(fill="gray90",
                                       colour = "black"),
        strip.text = element_text(size=18,
                                  colour="black",
                                  face = "bold"),
        plot.margin = unit(c(0.01,0.01,0.01,0.01), "cm"))

Opciones para elementos de theme

  • element_blank. Elemento en blanco.
  • element_line. Elemento de tipo línea.
  • element_rect. Elemento de tipo rectángulo.
  • element_text. Elemento de tipo texto.

Exportar gráfico

ggsave

Guarda por default la última gráfica agregada en el panel de gráficos.

Permite guardar imágenes en formato: png, eps, ps, tex, pdf, jpeg, tiff, bmp, svg. Si no se le indica lo toma del nombre del archivo.

ggsave(filename = "Gráfica_ggplot.jpeg",
       width = 18,
       height = 12,
       units = "cm",
       dpi = 300)
